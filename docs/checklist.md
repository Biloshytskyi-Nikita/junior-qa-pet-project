# Checklist

## Login
- Login page is opened successfully
- Username field is displayed
- Password field is displayed
- Login button is displayed
- User can log in with valid credentials
- User cannot log in with invalid credentials
- Error message is shown for invalid login
- Required fields validation works correctly

## Dashboard
- Dashboard is opened after successful login
- Dashboard widgets are displayed
- User profile menu is displayed
- Main menu is visible

## Navigation
- User can open PIM module
- User can open Admin module
- User can open Leave module
- Selected module is highlighted correctly

## PIM
- Employee List page opens successfully
- Search filters are displayed
- Employee table is displayed
- Add Employee button is visible

## Employee Search
- Search works with valid employee name
- Search works with partial input
- Search returns matching results
- Search handles no-result scenario correctly
- Reset button clears filters

## Logout
- User profile menu opens
- Logout option is available
- User can log out successfully
- Logged out user is redirected to login page
