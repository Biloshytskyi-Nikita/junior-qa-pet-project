# Test Plan

## 1. Objective
The objective of this test plan is to verify the core functionality of the OrangeHRM demo application and ensure that main user flows work as expected.

## 2. Scope

### In Scope
- Login functionality
- Dashboard access
- Navigation between modules
- PIM module
- Employee search
- Logout functionality

### Out of Scope
- Performance testing
- Security testing
- Mobile testing
- Cross-browser testing

## 3. Test Strategy
Testing will be performed manually using a checklist and detailed test cases.

The following types of testing will be applied:
- Functional testing
- Smoke testing
- Regression testing (basic)

## 4. Test Environment
- Browser: Google Chrome (latest version)
- OS: Windows
- Tools:
  - Chrome DevTools
  - Postman
  - Git

## 5. Entry Criteria
- Application is доступна
- Test account is available
- Requirements are defined

## 6. Exit Criteria
- All planned test cases are executed
- Critical bugs are identified and documented
- Test results are recorded

## 7. Risks
- Demo environment instability
- Limited functionality in demo version
- Pre-existing data affecting test results
