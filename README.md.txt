# Junior QA Pet Project

## Project Goal
This repository contains a manual QA pet project created for practicing web application testing skills relevant to a Junior QA Engineer role.

## Application Under Test
OrangeHRM Demo

## Scope
- UI testing
- Functional testing
- Basic API observation
- DevTools network analysis
- Test documentation
- Bug reporting
- Basic SQL practice
- Git workflow

## Tools Used
- Chrome DevTools
- Postman
- Git
- SQL
- Linux terminal
- Markdown documentation

## Repository Structure
- `docs/` — project documentation
- `test-cases/` — detailed test cases
- `bug-reports/` — bug reports
- `api/` — API-related checks
- `sql/` — SQL practice and validation queries
- `devtools/` — browser/network analysis
- `screenshots/` — supporting screenshots

## Expected Outcome
The project demonstrates practical QA skills required for a Junior QA Engineer position.
